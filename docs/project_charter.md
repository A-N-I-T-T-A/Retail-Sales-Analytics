# 📋 Project Charter

## Project Name
Retail Sales Analytics

---

## Project Description

This project analyzes retail sales data to uncover business insights related to sales performance, customer behavior, product performance, profitability, and regional trends.

The analysis will be performed using SQL, Python, Excel, and Power BI to simulate a real-world Data Analyst workflow.

---

## Business Problem

Retail companies generate large volumes of transactional data every day.

Management needs a clear understanding of:

- Revenue trends
- Product performance
- Customer purchasing behavior
- Regional sales performance
- Profitability

The goal is to transform raw sales data into actionable business insights that support strategic decision-making.

---

## Stakeholders

- CEO
- Sales Manager
- Marketing Team
- Finance Team
- Inventory Team

---

## Project Goals

- Analyze sales performance
- Evaluate profitability
- Identify top-performing products
- Understand customer purchasing behavior
- Compare regional performance
- Build executive dashboards
- Recommend business improvements

---

## Expected Deliverables

- SQL Business Analysis
- Python Data Cleaning
- Exploratory Data Analysis (EDA)
- Excel Dashboard
- Power BI Dashboard
- Business Report
- GitHub Documentation

---

## Success Criteria

The project should provide clear, data-driven recommendations that help management improve sales performance and profitability.
